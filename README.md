# Mon livre de recettes

Site web de recettes familial — React + Vite + GitHub Pages.

## Configuration

Copier `.env.example` en `.env.local` et remplir les valeurs.

## Développement

```bash
npm install
npm run dev
```

## Déploiement

Le déploiement est automatique sur push vers `main` via GitHub Actions.
Configurer les secrets GitHub Actions selon `.env.example`.
